Anime

The purpose of this project was to play around with an anime dataset I found on kaggle and find ways to analyze it. To accomplish all of this, I first pulled data a dataset that was provided to me courtesy of Kaggle and their “Anime Recommendation Database 2020”. I compared and analyzed the names, genre, type, rank, duration, score and other things that were provided in the dataset. After assembling the dataset. This website provides the source data, the visualizations that resulted from my analysis, and explanations and descriptions what was discovered while I analyzed and fiddled with the website.
